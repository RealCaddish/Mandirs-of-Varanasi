Shapefile to GeoJSON Converter

Step 1: Understanding Your Data

You have a shapefile containing temple/mandir data in the geoJSON/data/temple/ folder. The files include:

temple.shp - Main shapefile (63KB)
temple.dbf - Attribute data (945KB)
temple.shx - Shape index file (18KB)
temple.prj - Projection file (143B)
temple.cpg - Code page file (10B)

This appears to contain individual mandir locations with associated attribute data.

Step 2: Conversion Options

Since we can't use npm due to PowerShell execution policy, here are alternative methods:

Option A: Online Converter (Recommended)

Use one of these free online tools:

  • Mapshaper - Drag and drop your .shp file
  • Convertio - Simple online conversion
  • GeoJSON.io - Upload and convert

Option B: QGIS (Free GIS Software)

If you have QGIS installed:

1. Open QGIS
2. Add Vector Layer → Select temple.shp
3. Right-click layer → Export → Save Features As
4. Choose GeoJSON format
5. Save as temple.geojson

Option C: Python (if available)

If you can run Python:

pip install geopandas
python -c "import geopandas as gpd; gdf = gpd.read_file('geoJSON/data/temple/temple.shp'); gdf.to_file('geoJSON/data/temple/temple.geojson', driver='GeoJSON')"

Step 3: Expected Data Structure

Once converted, your GeoJSON file should contain:

  • Point features - Individual mandir locations
  • Attributes - Information about each mandir (deity type, size, location type, etc.)
  • Coordinates - GPS coordinates for each temple

Step 4: Integration with Dashboard

After conversion, the data will be automatically integrated into the dashboard with:

  • Individual mandir markers on the map
  • Deity distribution charts based on actual data
  • Detailed popups with mandir information
  • Spatial analysis of mandir patterns

Important Notes:

  • Make sure to save the converted file as temple.geojson in the geoJSON/data/temple/ folder
  • The dashboard is already set up to load this file automatically
  • If the file structure is different, we may need to adjust the data loading code
